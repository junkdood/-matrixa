# Description
You program a function void swap(int* a, int* b) to implement swapping a and b.  
Don't submit main function.  
# Input
Two num: a, b  
# Output
Two num:a, b after swapping a and b.  
# Example input
'1 2'  
# Example output
'Before swap: a=1, b=2  
swap a and b  
After swap:  a=2, b=1  '
